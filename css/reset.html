什么是 CSS reset?

不同浏览器对HTML标签都有默认的样式且存在差别;
浏览器的默认样式可能会给我们带来奇葩的展示效果;
所以我们需要覆盖浏览器的默认样式。
但是我们又不能一味的使用 * {/* style code*/} 将浏览器的默认样式全部覆盖, 我们需要创建 CSS reset 的 css 文件选择性的覆盖默认样式。
