# 碎碎念1️⃣

::: center

生活难免落俗,但浪漫不死.

傍晚微风轻拂,落日黄昏,你笑起来的时候真像好天气.

不用纠结,有没有书里说的永远,在一起的每分每秒都是永恒.

我不知道一生会遇到多少人,但希望我能成为你的铠甲,你的软肋,你的好运气,而你
则是我偷偷藏在心底的光.

我们之间就像海浪亲吻礁石,晚风拥抱月
亮,我看见归鸟,听见蝉鸣和你走到天光大亮.

学会忠诚,保持认真.
彼此成为更美好的人！

:::
